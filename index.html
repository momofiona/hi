title: 岁月搓一坨
date: 2015-08-29 22:03:06

tags:

昨晚下班回家，老婆告诉我说明天周末要带女儿去动物园。我吞了口饭：“啥~ 今天不是周二么”。

想想这些年日子过的满满当当，心里却还是空荡荡，今晚我想给个吐槽，为嘛自己上班总是浸在公司那些马马虎虎的项目里，像一颗坚硬的铆钉，做着自己份内的事情。 做好又怎样，做到最好又怎么样？脑子好使的都知道，写代码是整个项目中最容易的部分，是产品周期中最可控的环节。做到最好也只是写好了代码而已，而已，而已，离做好产品差得远呢。

吐完工作开始吐搭档。话说新搭档入职快满一年了，和刚走上社会的我非常相似，年轻，没有爱好，没有女朋友，没有运动，容易沉静在写代码的氛围中，解决了问题就会很有成就感，可以一天不喝水，吃饭不积极，下班不积极…… 作为过来人哥欣赏你，公司需要你，但你可能几年后才能发现，兢兢业业奉献了自己，自己因为没有把时间花在提升眼界上而进步缓慢。兄弟啊，哥哥提醒你很多次不要太注重细节，但貌似没起作用。哎，我的个人经验也不一定是适合你，就不唠叨你了。我这人不喜欢主动帮人，但是如果主动来找我，我很愿意讨论分享。

吐完搭档吐合作商。@#$%……哎呀妈呀真想吐了，恶心~下次和他有关的项目一个不接。

最后找个镜子吐槽下，帅哥变大叔，电视上演的那些幸福人生怎么就那么难呢？想云淡风轻你去道观啊，没钱就给我好好努力；那些贵人你点个头哈个腰去喝啊，机会总是藏在风浪中得去风浪中找啊……。
想起朋友的签名“耐得住寂寞才能守得住繁华”，继续加油。

昨晚下班突然决定回到家十几个红灯一灯不闯，一路上很多事情就想通了。一花一世界，有那么点感觉了。

瞎扯了这么久回到写本文的本意上来，其实我就是给客户改了个bug，写写博客记录下自己的功德。

客户网站上几处功能突然不能使用，检查后发现是网站从linux迁移到windows产生的问题

  • php document_root在IIS上为空问题
if(empty($_SERVER['DOCUMENT_ROOT']) && !empty($_SERVER['SCRIPT_FILENAME'])) { 
  $_SERVER['DOCUMENT_ROOT'] = str_replace( '\\', '/', substr($_SERVER['SCRIPT_FILENAME'], 0, 0 - strlen($_SERVER['PHP_SELF'])));
}
  • javaBridge缺包问题,增加一个基友包就解决了
java_autoload("sugonrsa.jar;commons-codec-1.10.jar");

团队协作中的Sublime

###代码检测

  • http://www.sublimetext.com/3 下载合适的portable version
  • https://nodejs.org/download/ 下载 node并且安装
  • https://packagecontrol.io/installation 按照说明配置sublime
  • sublime中ctrl+shift+p, 输入install,选择install Package
  • 安装 SublimeLinter
  • 安装SublimeLinter-jshint
  • 重启sublime
  • 在sublime右键，菜单最后有个SublimeLinter配置，配置前两项Lint Mode和Mark Style，分别表示触发事件和标记样式。
    *如果要修改默认js规则，可在项目目录下放配置文件，参考移动云目录下的.jshintrc 文件，（ps:团队项目的配置文件不要个人去修改）

#####sublime其他插件

  • HTML-css-JS Prettify ——代码格式化（团队必要）
  • Docblocker—— 注释
  • better Completion——代码提示
  • emmet——就是 zencoding
